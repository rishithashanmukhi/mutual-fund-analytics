# Mutual Fund Analytics

## Project Overview
This project analyzes mutual fund datasets and performs data ingestion, validation, and live NAV fetching using Python.

## Deliverables
- data_ingestion.py
- live_nav_fetch.py
- requirements.txt

## Datasets Used
- 01_fund_master.csv
- 02_nav_history.csv
- 03_aum_by_fund_house.csv
- 04_monthly_sip_inflows.csv
- 05_category_inflows.csv
- 06_industry_folio_count.csv
- 07_scheme_performance.csv
- 08_investor_transactions.csv
- 09_portfolio_holdings.csv
- 10_benchmark_indices.csv

## Features
- Load and validate CSV datasets
- Display shape and sample records
- Fetch live NAV data from MFAPI
- Basic data quality checks

## Technologies Used
- Python
- Pandas
- NumPy
- Requests
