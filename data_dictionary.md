\# Data Dictionary



01\_fund\_master.csv

\- amfi\_code : Unique fund code

\- scheme\_name : Fund scheme name

\- category : Fund category



02\_nav\_history.csv

\- date : NAV date

\- nav : Net Asset Value



08\_investor\_transactions.csv

\- investor\_id : Investor identifier

\- transaction\_type : SIP/Lumpsum/Redemption

\- amount : Transaction amount
